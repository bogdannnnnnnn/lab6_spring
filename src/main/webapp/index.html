Student Management API

Welcome to the Student Management API. This API allows you to manage students and their token balances.

API Documentation

Open Swagger UI

Available Endpoints

GET /api/v1/getStatus - Check application status
GET /api/v1/students - Get all students
GET /api/v1/students/{lastName}/{firstName} - Get specific student
POST /api/v1/students - Add new student (teacher only)
PUT /api/v1/students/{lastName}/{firstName}/tokens - Change tokens (teacher only)
DELETE /api/v1/students/{lastName}/{firstName} - Remove student (teacher only)

Usage Notes

  • All endpoints accept an optional 'role' query parameter (student or teacher)
  • Some operations require teacher role
  • Use Swagger UI for interactive API testing
